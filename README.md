# Max-Gym

## A responsive landing page for a gym created with HTML, CSS, Tailwind CSS, and JavaScript. This project showcases a modern, user-friendly interface designed to attract and engage gym members.

## Technologies Used

- **HTML**: For the webpage structure.
- **CSS**: Custom styles for additional designs.
- **Tailwind CSS**: For utility-first CSS and responsive styling.
- **JavaScript**: For interactivity and animations.
